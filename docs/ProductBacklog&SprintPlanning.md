## Product Backlog

- As a user, I want to create an account so that I can manage my tasks
- As a user, I want to log in so that I can access my data
- As a user, I want to create tasks so that I can stay organized
- As a user, I want to edit tasks so that I can update information
- As a user, I want to delete tasks so that I can remove completed items
- As a user, I want to assign priorities so that I can focus on important tasks
- As a user, I want to set due dates so that I can track deadlines
- As a user, I want to view all tasks so that I can see my workload
- As a user, I want reminders so that I do not forget tasks
- As a user, I want a dashboard so that I can see task summaries
