## 5. Responsible AI Usage

AI tools were used to assist in brainstorming, system design, and structuring the project. All AI-generated content was carefully reviewed and refined to ensure accuracy and alignment with project goals. Final decisions were made by the team.

---

### AI Usage Log

| AI Tool | Prompt Submitted | Purpose | Influence on Design |
|--------|----------------|--------|---------------------|
| ChatGPT | "Generate a full-stack web app idea for productivity and organization" | Brainstorming project idea | Helped define the overall concept and direction of the application |
| ChatGPT | "List features for a task management system MVP" | Exploring system features | Helped identify core MVP features such as task creation, dashboard, and reminders |
| ChatGPT | "Create example entities for a task management app" | Generating data models | Provided structure for User and Task entities |
| ChatGPT | "Design system architecture for a React and Spring Boot app" | Refining architecture | Helped organize frontend, backend, database, and API integration |

---

### Evaluation of AI Output
All AI outputs were critically evaluated and modified as needed. The team ensured that:
- The design decisions align with project requirements  
- The features are realistic and implementable  
- The architecture supports scalability and usability  

AI was used strictly as a support tool, and all final design choices were made by the team.
