 Product Backlog (User Stories)


As a user, I want to create an account so I can manage tasks

As a user, I want to log in so I can access my data

As a user, I want to create tasks so I can stay organized

As a user, I want to edit tasks so I can update details

As a user, I want to delete tasks so I can remove completed ones

As a user, I want to set due dates so I can track deadlines

As a user, I want to assign priorities so I can focus on important tasks

As a user, I want to view all tasks so I can see my workload

As a user, I want reminders so I don’t forget tasks

As a user, I want a dashboard so I can see task summaries
