## 2. External API Integration

### Selected API
The application will integrate the Google Calendar API (or a similar date/time API).

---

### Type of Data Provided
The API provides:

- Date and time information  
- Calendar events and scheduling data  
- Time-based reminders and event details  

---

### How the Application Will Use This Data
The application will use the external API to enhance task management and organization by:

- Syncing user-created tasks with real-world dates and times  
- Displaying upcoming deadlines based on current date/time data  
- Enabling time-based reminders for tasks  
- (Optional) Allowing users to integrate or import external calendar events  

---

### Relevance to Application Purpose
This API meaningfully supports the purpose of the application by improving time management and ensuring tasks are aligned with real-world schedules. Instead of simply storing tasks statically, the system becomes dynamic by incorporating live date and time data.

This enhances the user experience by helping users stay on schedule, meet deadlines, and manage their productivity more effectively.
